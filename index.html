SCORE KEEPER

To start this first you have to set the score using dropdown if score is greater than 5 than enter your score in textfield. Setting the names is optional. If you set the names the player 1 name button and player 2 name button will be replaced by the names you have entered respectively. After winning of anyone player the winner list will be updated in the winners list dropdown. Clicking reset button in that dropdown will reset the winners list. Clicking the reset button which is present at the bottom will reset the system.
